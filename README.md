# LPU Live Chat App

An Electron App for Lpu Live Chat Web Application as Desktop App

fixing `warning "xo > eslint-import-resolver-webpack@0.13.2" has unmet peer dependency "webpack@>=1.11.0"`

### Install 
~~~
yarn install
~~~
or 
~~~
yarn add webpack --dev
~~~

### Usage: 
~~~
yarn start
~~~
